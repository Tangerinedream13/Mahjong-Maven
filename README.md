Mahjong Maven was built for the PEO Chapter, a group of women who just want to play mahjong without dealing with spreadsheets. It handles all the logistics so you can focus on the game.

## What it does

- **Create a tournament** and set the date, location, and number of rounds
- **Add your players** and paste in the whole list at once
- **Generate rounds** and Maven seats the tables automatically each round
- **Enter scores** at the end of each round
- **See standings** with live rankings that update as scores come in
- **Leaderboard** with all-time rankings across every tournament your group has ever played

## Running it locally

Start the backend:
```bash
cd server && go run ./cmd/api/main.go
```

Start the frontend:
```bash
cd client && npm install && npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).
